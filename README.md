# Calculadora de Custos – Velas

Website simples (HTML + CSS + JavaScript) para somar custos de materiais (baseado na tua tabela).

## Como usar
1. Faz download / clone do repo
2. Abre o ficheiro `index.html` no browser
3. Marca/desmarca materiais para ver o total

## Editar preços
No `index.html`, cada preço está em `data-price="0.01"` (e o texto na coluna também).
Exemplo:
```html
<td data-price="0.10">0.10</td>
